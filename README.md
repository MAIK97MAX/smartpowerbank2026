# Semesterproject 2026 - Smart Powerbank

## Authors
Jukic Malik & Sayfutdinov David - HTL Rankweil

## Goal 
The goal of this project is to design and build a smart power bank with USB-C charging, battery management and system monitoring.

## Features
- USB-C charging
- OLED display for battery status and system information
- ESP8266 module for web interface
- Battery monitoring and protection
